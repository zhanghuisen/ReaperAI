ReaperAI v1.0.4 - 安装与使用说明
============================================

【安装步骤】

1. 复制 MCP_Server 、Scripts、UserPlugins 文件夹到Reaper根目录（建议 X:\REAPER）

2. 打开MCP_Server文件夹双击运行【第一步】安装依赖.bat，等待安装完成
   该脚本会自动准备 ReaperAI 私有 Python 环境；用户电脑没有 Python 也可以安装。

3. 双击运行【第二步】配置向导.bat，按中文提示完成配置
   路径支持正斜杠、反斜杠、带双引号和带空格路径；窗口不会因错误直接闪退。

4. 在 REAPER 中按 ? 打开 Action List，载入指令选择 X:\REAPER\Scripts\ReaperAI.lua 并运行

============================================

【使用步骤】

1. 在 REAPER 中运行 ReaperAI 脚本（按 ? 搜索 ReaperAI）

2. 在设置页签设置好LLM的API Key，并下划到本机功能检测页签，点击下方检测按钮来检测本机Action ID和本机Reaper API

3. 在对话页点击“执行模式”

4. 不开启执行模式只能对话（问为什么、怎么做）
    开启执行模式可通过发送文本命令操作Reaper

============================================

【注意事项】

• 不要求用户提前安装 Python。安装脚本会优先使用随包便携 Python，并从本地 wheels 离线安装核心依赖
• 第二步配置向导会优先使用随包/本地 Python 运行 UTF-8 中文界面，减少乱码和环境差异问题
• 确保端口 8765 未被占用
• 首次使用必须配置 LLM API Key
• 如需使用 ElevenLabs 语音/音效功能，在配置文件中取消注释 ELEVENLABS_API_KEY
